## 🚀 Flagship Project — InterviewForge

<div align="center">

[![InterviewForge](https://img.shields.io/badge/🎯_InterviewForge-AI_Mock_Interview_SaaS-F2CA50?style=for-the-badge&labelColor=0a0a0a)](https://ai-interview-coach-web.workers.dev)

</div>

> A production-grade AI SaaS built on cutting-edge Cloudflare infrastructure. Users get AI-generated interview questions, real-time scored answers via streaming SSE, shareable score cards, and company-specific prep — all with **zero cold starts**.

| Layer | Tech |
|-------|------|
| **Frontend** | Next.js 16 + React 19 + TypeScript + Tailwind CSS v4 |
| **Backend** | Python FastAPI on Cloudflare Workers (native Python runtime) |
| **AI** | Groq API · llama-3.3-70b · 300+ tokens/sec streaming |
| **Database** | Cloudflare D1 (SQLite) with 6-version migration history |
| **Auth** | Clerk v7 JWT (Next.js + Python middleware) |
| **Payments** | Stripe subscriptions + webhook idempotency |
| **Infra** | Cloudflare Workers · KV · R2 · OpenNext adapter |
| **Animations** | GSAP 3.14 (SplitText, DrawSVG, ScrollTrigger) + Motion 12 |
| **OG Images** | Satori on Edge runtime for viral share cards |
| **Video** | Remotion 4.0 for animated logo reveals |

**Features:** Real-time AI Q&A streaming · 4 interview types · Category score breakdown · Model answers · Resume parsing · Company-specific prep (KV-cached 24h) · Voice transcription · Viral share pages · Stripe tier enforcement (Free / Pro $19 / Premium $39)
